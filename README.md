# MarkSheet OCR & Grading System

ブラウザ上で動作するマークシート自動採点システムです。PDFをアップロードするだけで、解析から採点、CSV出力まで完結します。

## 機能
- **PDF入力**: マークシートのスキャンデータ（PDF）を読み込み
- **自動採点**: 1ページ目を正答（設定）として、2ページ目以降を自動採点
- **ブラウザ完結**: サーバーへのデータ送信なしで安全に処理
- **CSV出力**: 採点結果を一括ダウンロード

## 使い方（デプロイ方法）

GitHub Pagesでの公開が推奨です。

1. このリポジトリをGitHubにプッシュします。
2. GitHubのリポジトリ設定（Settings）を開きます。
3. **Pages** セクションへ移動します。
4. **Source** を `Deploy from a branch` に設定し、`main`（または`master`）ブランチを選択します。
5. 数分後、生成されたURLにアクセスするとシステムが利用可能です。

## ローカルでの実行

Node.jsがインストールされている場合：

```bash
npx serve .
```

ブラウザで `http://localhost:3000` にアクセスしてください。

## 注意事項
- 推奨ブラウザ: Chrome, Edge (最新版)
- PDF.jsのCDNを使用しているため、初回読み込み時はインターネット接続が必要です。
